eCDP/eSMART on WiiU - How to set it up.

There is a way to set up eCDP and eSMART 2.0 on a Wii U, this guide explains how to do it. Created by oscie

Preparations

To prepare for this, you need a Wii U with CFW (use this guide to install it, once done, make sure you have Tiramisu running as you follow the guide) and any DS game with Nintendo WFC connection (i.e. Mario Kart DS) as a Virtual Console game for online (eCDP ONLY)

Injects

  • McDonald's eCDP + eSMART 2.0 Injects
  • For Wii U, get the [WUP] versions
  • For CEMU, get the [WUD] versions (might be buggy)

Wii U Guide

Steps to Install

Step 1 - Moving Files

  1. Put your SD card into your computer.
  2. Find the "install" folder on the root.
  3. Extract the zip into the folder
  4. Put the SD card back in the Wii U

Step 2 - Installing the game

  1. Turn on your Wii U and open up the Homebrew Channel via Mii Maker
  2. Open up "WUP Installer GX" (if you don't have it, grab it here)
  3. Select eCDP/eSMART 2.0 and click Install
  4. Wait for it to install and exit the homebrew channel.

---- eCDP ONLY | Setting up Save ----

Step 3 - Getting the MAC Address

For this I will be using Mario Kart DS, other games can and will be different.

  1. Open up Mario Kart DS as stated in preparations.
  2. Tap on "Nintendo WFC" and ignore prompts.
  3. Tap on "Nintendo WFC settings" and go to options.
  4. Once there click on System Information and write down the MAC Address

Step 4 - Getting your serial code

  1. Click on this and type in your serial code.
  2. Set any Store ID and Store Management you want but make sure to write it down.
  3. Click on place offering and write down the serial code you are given.

Step 5 - Making the eCDP save

  1. Open up the game from your Wii U home menu.
  2. Click on the bottom option and type in the Store ID you set earlier.
  3. Press the button on the bottom of the keyboard
  4. Type in the Store Management ID you set earlier and press the button on the bottom of the keyboard.
  5. Next type in your serial code and once again press the button on the bottom of the keyboard
  6. Now this is where you set the manager ID and password, I recommend setting them both to ‚Äú482500‚Äù so they match up with everyone else's code.

Now you can explore eCDP to your heart's content!

CEMU Guide

Step 0 - Setting up CEMU

This guide assumes you have CEMU setup for use. If not, please follow the guide at cemu.hacks.guide

Step 1 - Preparing files

  1. Make sure you have [WUD]eSMART.zip downloaded somewhere.
  2. Extract this zip into a folder, any folder.
  3. Open the folder and make sure you have the "content", "meta" and "code" folders inside.

Step 2 - Installing

  1. Open up CEMU
  2. Move your cursor to the top left and go to "File".
  3. Click on "Install game title, update or DLC".
  4. Navigate through the file explorer until you find the folder we extracted our zip to.
  5. Open the "meta" folder and double click on "meta.xml".
  6. Wait until it is installed, and restart CEMU.

You should now have eSMART setup for use in CEMU! Just a warning - it can be quite buggy so do be aware.
