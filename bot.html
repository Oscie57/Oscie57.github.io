Oscie Bot 2

The Discord bot for shitpost and light moderation

How to set up:

    Step 1: Invite the bot to your server

    Step 2: Create a NEW invite to a public channel

    Step 3: DM that invite link to the bot

That should be it really, run o.help to see commands.
