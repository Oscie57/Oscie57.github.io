The Animal Crossing Dynamic Player (ACDP) was created last year, inspired by newleafradio and worked on for Python, with lots of new features and different game options.

Setup

Make sure you have Python installed! (This was tested on Python 3.8) Run pip install -r requirements.txt to install the required packages. Create a file called .env Inside that file, add AREA= and put your area afterwards. For example: AREA=Madrid, Spain Make sure you have .env and play.py in the same folder! Run python3 play.py (or py play.py on Windows) and it should run!
  • 1 - Run pip install -r requirements.txt to install the required packages.
  • 2 - Create a file called .env.
  • 3 - Inside that file, add AREA= and then the name of your city, for example: AREA=Madrid, Spain.
  • 4 - Once the dotenv is setup, run py play.py.
