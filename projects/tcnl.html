Did you know that there was a prequel to Tomodachi Life, called Tomodachi Collection? This mod replaces the 3DS game's music with the DS's.

Pre-requisites

In order to use this mod, you need a 3DS on the latest version, with the latest update for Tomodachi Life, a European version of the game (other region support coming soon), and to have Luma CFW installed.

If you do not have CFW installed, please do so at 3ds.hacks.guide.

Downloading the mod

In order to download the latest version of the mod, get the latest full release and (optional) latest update release(s) here.

Version information

Full releases are marked tcnl_vX.X.zip. These releases contain all files from previous versions. You can use this on its own. For example, tcnl_v0.3.zip is a full release.

Update releases are marked tcnl_vX.Xx.zip. These releases only contain new/updated files. You cannot use this on its own. In order to use these, you must get the latest full release before the update release and download all update releases before the one you are downloading. For example, tcnl_v0.3a.zip is an update release.

Title IDs

Each region of Tomodachi Life has a different Title ID representing it. Please refer to the list below to find the Title ID for your game's region.

  • 000400000008C300 - USA / Americas
  • 000400000008C400 - EUR / Europe and Australia
  • 000400000008C500 - JPN / Japan
  • 0004000000120C00 - KOR / South Korea
